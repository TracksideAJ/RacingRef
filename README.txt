Column update:

The sheet order changed. This build updates the fixed indexes to match:

- Event is now Column D, index 3.
- Legacy Track Name is Column E, index 4.
- Most Recent Track Name is Column F, index 5.
- Finish is Column G, index 6.
- Driver is Column J, index 9.
- SPEC is Column AU, index 46.
- Pole remains Column AY, index 50.

The minimalist black and white theme remains unchanged.
